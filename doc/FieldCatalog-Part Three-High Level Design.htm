src/main/java/com/itservicesdepot/validator

Contains all Utilities which are used across the system.

Contains all Constants defined and used across the system such as DAOConstant.java, ErrorCodeConstant.java

src/main/resources

Contains files such as resources.properties, dozer.properties

src/main/webapp

Contains all folders and files are used on Web side.

src/main/webapp/pages

Services Layer

This layer is called from Bean/Controller layer and access Database via DAO layer.

For every service, there is an interface file for it as ProductService below

package com.itservicesdepot.service;

import java.util.List;

import com.itservicesdepot.model.Product;

import com.itservicesdepot.model.Result;

public interface ProductService {

       public List<Product> getProducts();

       public Result addProduct(Product product);

       public int updateProduct(Product product);

       public int deleteProduct(Product product);

       public Product getProduct(int id);

       public Product getProduct(String name);

       public List<Product> searchByKeyword(String keyword) throws Exception;

}

And an implementation as ProductServiceImpl below

  1. Using Annotation @Service to define a Service.
  2. Using Annotation @Transactional (readonly=true|false) to declare a Transaction
  3. Using Annotation @PreAuthorize to apply Spring Security

package com.itservicesdepot.service;

@Service("productService")

@Transactional(readOnly = true)

public class ProductServiceImpl implements ProductService {

    @Autowired

    ProductDAO productDAO;

  

    @Autowired

    VersionDAO versionDAO;  

 

    /*** Annotation of applying method level Spring Security ***/

    @PreAuthorize("hasRole('ROLE_ADMIN')")

    @Transactional(readOnly = false)

    public Result addProduct(Product product) {

       Result result = new Result();

 

       result.setId(productDAO.addProduct(product));

       return result;

    }
